# AI Recon Tool

A high-precision reconciliation engine with AI-powered OCR and semantic column mapping.

## Architecture
- **Core:** Data processing and reconciliation logic (Pandas).
- **Handlers:** File ingestion (Excel, CSV, PDF) and OCR (Tesseract/EasyOCR).
- **API:** FastAPI for web-based access.
- **Desktop:** PySide6 for Windows application.
